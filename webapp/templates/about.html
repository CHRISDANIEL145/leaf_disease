LGNM

Leaf-Graph Neural Morphing

Extracts vein structures from leaf images and processes them using Graph Neural Networks. Combines visual CNN features with topological graph features for enhanced disease detection.

MobileNetV3 GNN Vein Analysis

ADSD

Adaptive Disease Signature Diffusion

Uses signature-based augmentation to enhance disease pattern recognition. Lightweight architecture optimized for speed while maintaining high accuracy.

EfficientNet-B0 Signature Learning Fast Inference

S-ViT Lite

Spectral Vision Transformer

Combines RGB with pseudo-NIR spectral information using spectral attention. Vision Transformer architecture for global context understanding.

ViT Architecture Pseudo-NIR Spectral Attention

Explainable Causal Attribution Module

Provides interpretable predictions through causal reasoning. Generates attention maps showing which leaf regions influenced the diagnosis.
